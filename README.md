# mh2

## Developing:

```
npm install -g esy
git clone git@github.com:Faliszek/mh2.git
esy install
esy build
```

## Running Binary:

After building the project, you can run the main binary that is produced.

```
esy run
```

## Rebuild project && run:

```
esy pesy && esy build && esy run
```

## Running Tests:

```
# Runs the "test" command in `package.json`.
esy test
```
